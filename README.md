# Winterorbit API

#### requirements

  - Docker
  - docker-compose
  - OPTIONAL: for runing in Python virtual env - Python 3.7+ & pipenv
# How to run

  In Docker
```sh
$ sudo docker-comppose up --build
```
In Python virtual env - 
```sh
$ pipev shell
$ python manage.py runserver
```

#### Configuration

Service configuring via .env-file
